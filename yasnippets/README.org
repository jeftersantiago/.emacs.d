** Structures
   - =^= : ^{=cursor=}
   - =exp= : e^{=cursor=}
   - =df= : \frac{d =cursor=}{d =cursor=}
   - =pd= : \frac{\partial =cursor=}{\partial =cursor=} 
   - =dintegral=
     #+begin_src latex
       \int_{=lower=}^{=upper=}{=cursor=}
     #+end_src
   - =lim= : lim_{=from= \to =to=}{=cursor=}
   - =curl= or =rot=: $\nabla \times$ =cursor=
   - =div=: \( \nabla \cdot  \) 
   - =;v= : \vec{=cursor=}
   - =begin= : \begin{=name=} =cursor= \end{=name=}
   - =eq= :
     #+begin_src latex
          \begin{equation}
               \boxed{=cursor=}
               \label{=cursor=}
           \end{equation}
     #+end_src 
   - =$b= :
     #+begin_src latex
